## [添加注释](https://blog.jetbrains.com/clion/2016/05/keep-your-code-documented/)
1. 在函数名称上面一行输入 /*！然后回车，即可自动生成注释。
2. 在函数名称上面一行输入 /** 然后回车，即可自动生成注释。
3. 在函数名称上面一行输入 /*** 然后回车，即可自动生成注释。
4. 在函数名称上面一行输入 /// 然后回车，即可自动生成注释。


## 使用

mkdir build
